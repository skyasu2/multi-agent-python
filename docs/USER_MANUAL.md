# 📘 PlanCraft 사용자 매뉴얼

PlanCraft Agent를 사용하여 전문적인 비즈니스 기획서를 작성하는 방법을 안내합니다.

---

## 1. 시작하기 (Getting Started)

### 1.1 실행 방법
웹 브라우저에서 다음 주소로 접속합니다. (로컬 실행 시)
```
http://localhost:8501
```

### 1.2 화면 구성
1.  **좌측 사이드바**: 설정 메뉴, 이전 기록, 도움말.
2.  **메인 채팅창**: AI와 대화하며 기획서를 만드는 공간.
3.  **상태 표시줄**: 현재 어떤 에이전트가 작업 중인지 보여주는 타임라인.

---

## 2. 기본 사용법 (Basic Usage)

### 2.1 기획서 작성 시작하기
채팅 입력창에 만들고 싶은 서비스 아이디어를 입력하세요.

*   **짧고 단순하게**: "배달 앱 만들어줘"
    *   👉 **AI 제안 모드**: AI가 구체적인 컨셉(타겟, 차별점 등)을 제안하고, 사용자 확인을 거쳐 진행합니다.
*   **구체적으로**: "직장인을 위한 점심 메뉴 추천 앱인데, 매일 11시에 알림이 오고, 팀원들과 투표할 수 있는 기능을 넣어줘."
    *   👉 **Fast Track**: AI가 바로 의도를 파악하고 기획서 작성을 시작합니다.

### 2.2 중간 확인 절차 (Human-in-the-Loop)
작업 도중 AI가 확실하지 않은 부분이 있으면 작업을 **일시 정지(Pause)**하고 질문합니다.
*   화면에 표시된 **옵션 버튼** 중 하나를 선택하거나,
*   **직접 입력**을 통해 원하는 방향을 알려주세요.
*   ⚠️ **주의**: 일정 시간(기본 1시간) 동안 응답이 없으면 기본 설정으로 자동 진행됩니다.

---

## 3. 고급 기능 (Advanced Features)

### 3.1 품질 모드 설정 (Quality Presets)
사이드바에서 생성 모드를 선택할 수 있습니다.

| 모드 | 속도 | 특징 | 추천 상황 |
| :--- | :--- | :--- | :--- |
| **🚀 Fast** | 빠름 (<1분) | 핵심 내용 위주, 차트 미포함 | 아이디어 스케치, 브레인스토밍 |
| **⚖️ Balanced** | 보통 (2~3분) | 균형 잡힌 내용, 표준 차트 | 일반적인 기획서 작성 (**기본값**) |
| **💎 Quality** | 느림 (3~5분) | 심층 분석, 복잡한 시각화, 교차 검증 | 최종 보고서, 투자 제안서 |

### 3.2 파일 업로드
기존에 작성한 메모나 참고 자료(PDF, TXT)가 있다면 업로드하세요.
AI가 해당 내용을 분석하여 기획서에 반영합니다.

## 4. 자주 묻는 질문 (FAQ)

**Q. 기획서가 마음에 들지 않아요.**
A. 1차 초안이 나온 후, "수익 모델을 더 구체적으로 써줘"와 같이 피드백을 입력하세요. Refiner 에이전트가 이를 반영하여 문서를 수정합니다. (최대 3회)

**Q. 실제 시장 데이터인가요?**
A. 네, Market Agent가 실시간 웹 검색(Tavily)을 통해 최신 시장 규모와 경쟁사 정보를 수집합니다. 단, AI의 특성상 일부 오차가 있을 수 있으므로 더블 체크를 권장합니다.

**Q. 일반적인 질문도 되나요?**
A. 네, "안녕하세요"나 "오늘 날씨 어때?" 같은 가벼운 대화도 가능합니다. 이 경우 기획서는 생성되지 않고 챗봇처럼 답변합니다.

---

## 5. 문제 해결 (Troubleshooting)

*   **멈춤 현상**: 브라우저를 새로고침(F5) 하세요. 작업 내역은 남아있지 않을 수 있습니다.
*   **에러 메시지**: "Max retries reached" 메시지가 뜨면, 입력 시간이 초과되어 기본값으로 진행된 것입니다. 결과를 확인 후 수정 요청하세요.
