# TAM/SAM/SOM 작성 가이드

## 개요

시장 규모 분석은 기획서의 필수 요소입니다. 단일 수치("시장 규모 $50B")만 제시하면 **FAIL** 처리됩니다.
반드시 TAM → SAM → SOM 3단계로 분석하세요.

---

## 🚨 치명적 오류 (FAIL 조건)

### ❌ 잘못된 예시들

1. **단일 수치만 언급**
   > "위치 기반 서비스 시장은 585억 달러입니다"
   
   문제: TAM/SAM/SOM 구분 없음 → **FAIL!**

2. **SOM = 1년차 매출 (100% 달성)**
   > "SOM: 10억원, 1년차 매출: 10억원"
   
   문제: 초기 스타트업이 1년차에 SOM 100% 달성은 비현실적 → **FAIL!**

3. **무관한 시장을 TAM으로 설정**
   > "TAM: 웨어러블 시장 $70B" (뇌파 명상 앱인데)
   
   문제: 직접 관련 시장이 아님 → **FAIL!**

---

## ✅ 올바른 작성법

### TAM (Total Addressable Market) - 전체 시장

- **정의**: 서비스가 속한 **직접 시장**의 글로벌 규모
- **단위**: 달러 ($B, $M)
- **주의**: 우리 서비스와 직접 관련된 시장이어야 함!

**올바른 예:**
```markdown
- **TAM (Global)**: 위치 기반 피트니스 앱 시장 $12.5B (2026년, 출처: Grand View Research)
```

**잘못된 예:**
```markdown
- **TAM**: 전체 IT 시장 $5T (너무 광범위!)
- **TAM**: 웨어러블 시장 $70B (러닝 앱과 무관)
```

### SAM (Serviceable Addressable Market) - 접근 가능 시장

- **정의**: TAM 중 우리가 **실제 접근 가능한** 지역/세그먼트
- **단위**: 원화 (조 원, 억 원) - 국내 시장 기준
- **주의**: TAM의 직접적 부분집합이어야 함!

**올바른 예:**
```markdown
- **SAM (국내)**: 국내 러닝/피트니스 앱 시장 5,000억 원 (출처: 한국콘텐츠진흥원, 2025)
```

**잘못된 예:**
```markdown
- **SAM**: TAM의 14% = $10B (근거 없는 비율!)
- **SAM**: 국내 전체 앱 시장 10조 원 (너무 광범위)
```

### SOM (Serviceable Obtainable Market) - 획득 가능 시장

- **정의**: 1년차에 **현실적으로 획득 가능한** 시장 점유 목표
- **단위**: 억 원
- **주의**: 초기 스타트업 기준 10억~100억 원이 현실적!

**올바른 예:**
```markdown
- **SOM (1년차 목표)**: 50억 원
  - 계산: MAU 50,000명 × 전환율 10% × 연간 객단가 10만원 = 5,000명 × 10만원 = 5억원
  - 또는: 월 구독 5,000원 × 유료 사용자 10,000명 × 12개월 = 6억원
```

**잘못된 예:**
```markdown
- **SOM**: $1B (비현실적! 초기 스타트업이 1조원?)
- **SOM**: 10억원, 1년차 매출: 10억원 (달성률 100% = 비현실적!)
```

---

## 📊 SOM vs 1년차 매출 Gap 설명 (필수!)

SOM과 1년차 매출이 다르면 **반드시 Gap 극복 전략**을 설명하세요.

### 필수 포함 요소

1. **달성률**: 1년차 매출 / SOM × 100%
2. **Gap 극복 전략**: 2년차, 3년차 계획

### 올바른 예시

```markdown
### SOM vs 1년차 매출 Gap

| 구분 | 금액 | 달성률 |
|------|------|--------|
| SOM (목표) | 50억 원 | - |
| 1년차 매출 | 5억 원 | 10% |

**Gap 분석:**
- 1년차에는 초기 고객 확보와 제품 검증에 집중
- 2년차: 마케팅 확대 + 바이럴 캠페인 → 달성률 30% (15억원)
- 3년차: B2B 기업 웰니스 진출 → 달성률 60% (30억원)
- 4년차: SOM 100% 달성 목표
```

### 잘못된 예시

```markdown
- SOM: 10억원
- 1년차 매출: 10억원

(Gap 설명 없음! → FAIL)
```

## 📊 성장률 현실성 체크

분기별 성장률이 너무 높으면 근거가 필요합니다.

| 분기 | 매출 | 성장률 | 평가 |
|------|------|--------|------|
| Q1 | 1억 | - | - |
| Q2 | 2억 | 100% | ✅ 합리적 |
| Q3 | 5억 | 150% | ⚠️ 근거 필요 |
| Q4 | 15억 | 200% | ❌ 비현실적 (근거 없으면) |

**200% 이상 성장 시 반드시 근거 명시:**
- 투자 유치 완료 (시리즈 A 등)
- 대규모 마케팅 캠페인 집행
- B2B 대형 계약 체결
- 바이럴 마케팅 성공 사례

## 자주 하는 실수 TOP 5

1. **시장 규모 = TAM 하나만 언급** → SAM, SOM 필수!
2. **SOM = 1년차 매출** → Gap 설명 필수!
3. **TAM이 우리 서비스와 무관한 시장** → 직접 시장으로!
4. **출처 없는 시장 규모** → 반드시 출처 명시!
5. **비현실적 성장률(300%+)** → 근거 필수!
